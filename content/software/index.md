        ## Bubble Analyser

        [Bubble Analyser](https://github.com/p-quintanilla/BubbleAnalyser) is an open-source MATLAB tool for bubble size measurement from images, developed for froth flotation research. It provides automated image segmentation and size distribution analysis.

        - **GitHub:** [p-quintanilla/BubbleAnalyser](https://github.com/p-quintanilla/BubbleAnalyser)
        - **Publication:** [Minerals Engineering (2022)](https://www.sciencedirect.com/science/article/pii/S0892687522001078)
