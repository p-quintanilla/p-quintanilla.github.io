        ## Bubble Analyser

        [Bubble Analyser](https://www.bubbleanalyser.com/) is an open-source tool for bubble size measurement from images, developed for froth flotation research. It provides automated image segmentation and size distribution analysis.

        - **Website:** [www.bubbleanalyser.com](https://www.bubbleanalyser.com/)
